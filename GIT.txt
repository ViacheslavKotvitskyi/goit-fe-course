

    git remote show origin  - показывает путь к удаленному репозиторию т еше инфу
    git clone адресс  - клонирует рипозиторий еа компьютер


    git log --graph  - выводит список коммитов
    git diff - показывает конкретные изменения 

-Создание ветки
      git branch имя ветки - создает ветку
      git checkout имя ветки - переход в ветку
      git branch - показывает в какой ветке мы находится
      (git checkout -b <имяветки> является шорткатом для git branch <имяветки> за которым идет  git checkout <имяветки>.)
